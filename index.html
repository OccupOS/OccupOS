What is OccupOS?

OccupOS is a complete software solution for environmental monitoring that has been designed from the ground up to be abstracted from the hardware. Whether you already have sensor systems in place, or are exploring the possibilities for adding them, OccupOS can improve your existing options by leaps and bounds.

The project is still in its early stages of development, but you can grab our code and test it in a real environment already.

Flexible

OccupOS is not tied to any particular hardware. If we don't already support a device, you can implement our simple JSON interface and begin sending data from new sensors (we appreciate pull requests for any improvements you might make!).

Default Hardware Support

We do support a limited (but expanding) set of hardware by default, namely:

  • Netduino Pro (with Weather Shield and WiFi Shields).
  • Gadgeteer Spider Fez (with light sensor module).
  • Microsoft Kinect.
