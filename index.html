What is OccupOS?

OccupOS Occupancy Operating System intelligently uses various (low-powered) sensors to increase energy efficiency, environment quality and track precise occupancy in large buildings in an open, extensible, efficient and social way. It has been designed from the ground up to be abstracted from the hardware, so whether you already have sensor systems in place, or are exploring the possibilities for adding them, OccupOS remains cost-effective.

v.1.0.0

We've come a long way since starting the project and though the source code has been available from the start, it may have not been immediately clear how to build or deploy it in your own environment.

With our v1.0.0 release, we're happy to report that we now have detailed documentation on both deployment and contribution.

Current Hardware Support

We support a significant set of sensors by default, namely:

  • FEZ Spider Mainboard, with:
    • WiFi RS21 Module
    • LightSense Module
  • Netduino Plus 2, with:
    • Arduino WiFi Shield
    • Melexis MLX90620
    • RN-XV WiFly Module
    • EtherMania WeatherShield
  • Microsoft Kinect

Note: Bold devices are required to run their associated boards.

Flexible

One of OccupOS's key aims is hardware independence. If we don't already support your device, either let us know or submit a pull request - it's a breeze to add the functionality yourself.
